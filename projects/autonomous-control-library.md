# Autonomous Control Library

**Context:** Developed middleware components for autonomous control used in off-highway machines while at Danfoss Power Solutions.

**Overview:** Implemented middle-layer software blocks including LiDAR drivers, sensor fusion stubs, and control interfaces wrapped in PLUS+1 GUIDE for embedded Linux controllers.

**Responsibilities:**
- Designed and implemented driver interfaces for LiDAR hardware.
- Integrated drivers with the Autonomous Control Library and compliance modules.
- Wrote unit and integration tests, and participated in system verification.

**Tech stack:** C, C++, PLUS+1 GUIDE, embedded Linux, ROS

**Outcome / Impact:** Delivered production-ready middleware that improved perception integration and reduced integration time for machine autonomy features.
