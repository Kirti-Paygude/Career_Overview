# Load Sensor Development

**Context:** Work at HUSCO Hydraulics on sensor hardware and embedded firmware.

**Overview:** Participated in load sensor product development from component selection through production and certification, including embedded firmware and calibration.

**Responsibilities:**
- Selected sensing components and designed firmware for sensor signal processing.
- Implemented automated sensor calibration routines and test harnesses.
- Created DFMEA inputs and technical documentation for production handoff.

**Tech stack:** Embedded C, IAR Embedded Workbench, CAN (J1939)

**Outcome / Impact:** Contributed to a production-ready load sensor with automated calibration, improving reliability and manufacturability.
