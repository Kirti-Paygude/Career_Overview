# UPS Control & Monitoring

**Context:** R&D at Magnaflux Systems and early embedded projects.

**Overview:** Developed embedded assembly and higher-level software for UPS control and monitoring, including test and simulation tools.

**Responsibilities:**
- Wrote embedded firmware in assembly for UPS control loops.
- Built simulation and test software in NI LabVIEW.
- Performed hardware testing and validation.

**Tech stack:** Assembly, NI LabVIEW, MPLAB

**Outcome / Impact:** Improved UPS reliability through better control logic and test automation.
