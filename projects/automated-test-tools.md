# Automated Test Tools for Rail Electronics

**Context:** Developed at Sphinx Worldbiz (contract to Knorr-Bremse Technology Center India).

**Overview:** Built automated test frameworks and LabVIEW OOP modules to accelerate verification of rail electronics subsystems.

**Responsibilities:**
- Designed modular test runner and reusable test components.
- Integrated REST APIs for dynamic test generation and reporting.
- Streamlined version control and CI processes for test assets.

**Tech stack:** LabVIEW, REST APIs, Python (support scripts), TortoiseSVN

**Outcome / Impact:** Reduced manual testing time, increased test coverage, and improved release confidence.
